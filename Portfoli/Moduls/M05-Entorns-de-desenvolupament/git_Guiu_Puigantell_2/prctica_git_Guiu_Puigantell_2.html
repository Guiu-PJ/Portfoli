El primer que has de fer es activar el git.

Per a fer això anem al projecte fem botó dret i seleccionem la següent opció:

Apareixara aquesta finestra on seleccionem el botó ok

Ara necessitarem guardar l'estat dels fitxers en el git.

Anirem en el bloc que volguem fer servir, apretem botó dret/Git/Commit

Ens demana un comentari, és important per a saber en quin moment vas fer el commit per després no tenir problemes si tens un treball gran i no saps en quin commit necessites cargar.

Ara per fer la prova de que esta activan el git editem un dels fixers

I podem veure com el editat ara surt de color blau perquè detecta que hi ha hagut canvis que no están en el ultim commit que hem fet, d’igual manera els que no tenen canvis surten de color negre.

En l'apartat del history podrem veure quan hem fet els canvis y els commits realitzats

Fent doble click en el commit que necessitem podrem veure el fitxer en el moment on hem fet el commit i el compara amb el actual així podem veure els canvis.

si apratem botó dret podem donar li a revert i fer que el fitxer torni a l'estat del commit que tinguem seleccionat

Amb botó dret/Git/Diff/Diff To podem comparar el fitxer amb el commit que nosaltres seleccionem

Seleccionem en quin commit el volem comparar

I els compara

Per a poder fer servir el git en remot, primer accedim a l apagina del Git

Un cop dins creem un nou projecte

Posem un nom i si es vol una descripció al projecte

Anem a clone i copiem la url per enllaçar-lo al netbeans

Anem a botó dret/Git/Remote/Push i enganxem l’enllaç que hem copiat abans

Enganxem la url on posa Repository URL

Marquem la primera i única opció

Seleccionarem l'opció de Yes perque sino no podrem fer servir el git en remot

Anem a la pàgina del git i canviem el main per el master

I quan acabi de cargar veurem el nostre commit en el git remot

Per crear branques anem a Team/Branch/Create Branch

posem el nom que millor recordem i creem la branca

Si anem a Team/Branch/Switch to branch

Podrem veure les branques que tenim creades i cambiar entre elles

Per poder ajuntar les branques podem fer un Marge

Per això anem al git remot i a l'apartat de Marge requests

Triem quines 2 branques volem fer servir, en aquest cas sera la master i la main

Posem el nom que vulguem, per exemple marge 1 i les demés opcions les deixem per defecte, comprobara tots el fitxers i si tot està correcte podrem fer el marge
